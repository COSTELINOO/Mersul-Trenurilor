1. Instalarea pkg-config

1.1 Pentru Ubuntu/Debian:

sudo apt update
sudo apt install pkg-config


1.2 Pentru Fedora:

sudo dnf install pkg-config


1.3 Pentru Arch Linux:

sudo pacman -S pkg-config





2. Instalarea libxml2 (librăria XML)

2.1 Pentru Ubuntu/Debian:

sudo apt install libxml2-dev


2.2 Pentru Fedora:

sudo dnf install libxml2-devel


2.3 Pentru Arch Linux:

sudo pacman -S libxml2



3. Instalarea pthread (librăria pentru thread-uri)

3.1 Pentru Ubuntu/Debian:

sudo apt install glibc-devel


3.2 Pentru Fedora:

sudo dnf install glibc-devel


3.3 Pentru Arch Linux:

sudo pacman -S glibc
